PIXEL MAKER
A browser-based pixel art editor with a retro game aesthetic. No installs, no sign-ups — just open the file and start drawing.

What You Can Do
Drawing Tools

Pencil — draw pixel by pixel with your primary color
Eraser — remove pixels cleanly
Bucket Fill — flood-fill any area with a single click
Color Picker — sample any color already on the canvas
Right-click anywhere to paint with your secondary color

Mirror Drawing

Horizontal mirror — everything you draw is reflected left-to-right simultaneously
Vertical mirror — reflected top-to-bottom
Combine both for perfect 4-way symmetry — great for icons, sprites, and mandalas

Canvas

Starts at 16x16 by default
Resize to any custom width and height up to 128x128 at any time — existing artwork is preserved
Zoom in and out across 9 levels to work comfortably at any scale

Colors

32-color built-in palette to get started instantly
Full color wheel for picking any color you want
Hex code input — paste any hex value directly
Primary and secondary color slots — swap them with one click
Add any custom color to your palette and reuse it anytime

Save and Load

4 save slots — save your work and come back to it later without losing anything
Each slot shows the canvas size and date saved
Saves persist in your browser — they survive closing and reopening the app

Import

Import any image from your device
Choose how to pixelate it — resize it to 16x16, 32x32, 64x64, or a fully custom size
Instantly turns photos, logos, or references into editable pixel art

Export

Export your artwork as a clean PNG file — pixel-perfect, no grid lines, no background artifacts
The exported file matches your canvas exactly (a 32x32 canvas exports a 32x32 PNG)

Undo and Redo

Full undo and redo history — step back through up to 60 actions
